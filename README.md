# Tutorial Programs for WebGPU in Rust
Some simple programs to get started with WebGPU concepts. Uses the Winit and WebGPU crates.

I originally set up this code for some of my own projects I was working with, but, perhaps ironically, that means that this WebGPU project can't target web builds. I don't think an enormous difference in configuration, so I'll look into adding it in the future. For now it'll have to be what it is or it would never get released!

I hope it's helpful regardless!
